# EOS-Test-Cave

Automated EOS Testing scripts.

These are only first steps in creating test scripts scenarios.

## Dependencies

The test runner requires `jq` and `bc`.

## Setup

- Clone the repo
- Start the wallet with `cd wallet && ./start.sh`
- Run tests with `./start.sh`
